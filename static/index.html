Countries API

Countries API

This API was develop on sinatra and ruby.

List of Endpoints

/countries
Get all countries.
/countries/name/{name}
Get country by name.
/countries/code/{code}
Get country by code.
/countries/capital/{capital}
Get country by capital.
/countries/callingcode/{callingcode}
Get country by calling code.
/countries/currencycode/{currency}
Get country by currency.
/countries/language/{language}
Get country by language.
/countries/region/{region}
Get country by region.
/countries/subregion/{subregion}
Get country by subregion.
